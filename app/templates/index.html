About:

Project "Rare Hashes" was created to find rare repeated values ​​in hashes.

The rarity is to find a repeating pattern (0 or 1) at the beginning or end of the hash. The minimum hash that is accepted for saving in the database must have no less than 25 repeating characters.

You can save this hash using a word that turns into this hash. On the server side, it is checked whether the hash is really rare and if successful, it is saved to the database along with the (optional) nickname

Methods to write your hash:

The first method is to click on the button below. You will be taken to a form that you need to fill out to enter the rare hash into the database.

The second method is to download a bot from my github to automatically find and send hashes

Last method is to put POST method with properiate json content to .../write.

Example json content:

{
    "word": "pRkKpnrndc1^z!T:6@Q=\"y9j0a#p=S=;",
    "hashType": "sha256",
    "user": "YAr"
}
